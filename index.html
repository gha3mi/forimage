fpm dependency

If you want to use ForImage as a dependency in your own fpm project, you can easily include it by adding the following line to your fpm.toml file:

[dependencies]
forimage = {git="https://github.com/gha3mi/forimage.git"}

How to run examples

Clone the repository:

You can clone the ForImage repository from GitHub using the following command:

git clone https://github.com/gha3mi/formatmul.git

cd formatmul

To run the examples using fpm, you can use response files for specific compilers:

For Intel Fortran Compiler (ifort):

shell fpm @ifort

For Intel Fortran Compiler (ifx):

shell fpm @ifx

For NVIDIA Compiler (nvfortran):

shell fpm @nvfortran

For GNU Fortran Compiler (gfortran):

shell fpm @gfortran

API documentation

The most up-to-date API documentation for the master branch is available here. To generate the API documentation for ForImage using ford run the following command:

ford ford.yml
